# sau-2025
Project: alien stage

This project is a simple webpage about alien stage by VIVINOS.
The page includes:
• a header and navigation menu
• an "About" section
• a table of top
• a gallery using CSS Grid
• a footer with media licenses

Used tags:
<!DOCTYPE html>, <html>, <head>, <meta>, <title>, <link>,
<header>, <nav>, <a>, <main>, <section>, <h1>–<h2>, <p>,
<table>, <caption>, <tr>, <th>, <td>, <img>, <footer>

Used CSS:
• external stylesheet (styles.css)
• Grid layout for gallery
• Flexbox for navigation
• styling for table, header, text, layout
